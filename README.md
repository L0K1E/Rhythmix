# ğŸ¶ Rhythmix

A modern web-based **drum machine and synth sequencer**, built with **Next.js, TypeScript, TailwindCSS, and Web Audio API**.
This project is designed for **producers, beatmakers, and hobbyists** who want a fast, clean, browser-based tool for making rhythms and melodies.

---

## âœ¨ Features

* **Step Sequencer**
  Add, remove, and control multiple tracks with a grid-style interface.

*  **Drum Voices**
  Includes a wide range of drum sounds (kicks, snares, hats, claps, percussion, FX, world instruments).

*  **Synth Voices**
  Multiple categories of synth tones (bass, leads, pads, plucks, FX).

*  **Track Management**

  * Rename projects (double-click project name)
  * Individual track volume control
  * Per-track clear/reset
  * Remove or add new voices anytime

*  **Randomizer**
  Generate musical variations automatically:

  * Randomly add 6â€“10 tracks
  * Smart voice selection (kicks, hats, bass, FX)
  * Auto-balanced volumes per track
  * Place beats in musically pleasing patterns

*  **Save & Load**

  * Save project as JSON
  * Load project back from file
  * Project name used as filename

*  **Export Options**

  * Export as audio (WAV/MP3 planned)
  * Export as project JSON

*  **Transport Controls**
  Music-player style UI with play/pause, timeline slider, volume, and icons.
